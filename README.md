# Responsive Search Box 🔍

## Project Description

This project is a **Responsive Search Box UI Component** built using **HTML and CSS only**.
The search box expands smoothly when the search icon is clicked and collapses when clicked again.

The interaction is created using the **CSS `:checked` pseudo-class** with a hidden checkbox input. No JavaScript is used in this project.

This project is useful for practicing **modern UI design and advanced CSS selectors**.

---

## 🚀 Features

* Responsive design
* Smooth width animation
* Interactive UI without JavaScript
* Clean and simple layout
* Font Awesome search icon
* Beginner friendly frontend project
